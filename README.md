# Mente Ativa

## Descrição

Mente Ativa é um aplicativo web desenvolvido como atividade de extensão
do curso de Análise e Desenvolvimento de Sistemas.

O objetivo é estimular funções cognitivas por meio de atividades
simples, utilizando imagens, sons e instruções por áudio.

## Funcionalidades

-   Jogo da Memória
-   Jogo dos Sons
-   Jogo do Troco
-   Resultado Final com pontuação

## Tecnologias

-   HTML5
-   CSS3
-   JavaScript

## Estrutura do Projeto

    css/
    js/
    imagens/
    audios/
    index.html

## Como executar

1.  Baixe os arquivos do projeto.
2.  Mantenha a estrutura das pastas.
3.  Abra o arquivo `index.html` em um navegador.
